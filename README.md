# 好好音乐仓库

一个基于 GitHub 的音乐存储和分享项目，按心情和场景分类整理优质音乐。

## 项目简介

好好音乐仓库是一个用于收藏和分享音乐的开源仓库，我们相信好的音乐应该被更多人听到。项目按照使用场景和情绪状态对音乐进行分类，方便你在不同心情下快速找到适合的音乐。

**仓库地址**: https://github.com/pkoiuu/haohao-music

## 如何提交歌曲

我们欢迎大家贡献优质音乐！请按照以下步骤提交：

### 1. Fork 仓库

点击页面右上角的 **Fork** 按钮，将仓库 Fork 到你自己的账号下。

### 2. 克隆仓库

将 Fork 后的仓库克隆到本地：

```bash
git clone https://github.com/你的用户名/haohao-music.git
cd haohao-music
```

### 3. 添加音乐文件

根据歌曲风格，将音乐文件放入对应的分类目录中：

- 适合放松的 → `休闲洗澡/`
- 欢快活力的 → `开心活泼/`
- 深夜抒情的 → `深夜eom/`

**文件命名建议**: `歌曲名-歌手名.扩展名`  
例: `晴天-周杰伦.mp3`

**支持格式**: mp3、flac、wav 等常见音频格式

### 4. 提交更改

```bash
git add .
git commit -m "feat: 添加歌曲 [歌曲名-歌手名]"
git push
```

### 5. 创建 Pull Request

回到 GitHub 页面，点击 **Compare & pull request** 按钮，填写 PR 说明后提交。
